# העלאה ל-GitHub Pages

1. צור Repository חדש או פתח את הקיים.
2. העלה את כל התיקייה YASOU-App-PRO.
3. ודא ש-index.html נמצא בשורש.
4. GitHub → Settings → Pages.
5. Source: Deploy from branch.
6. Branch: main / root.
7. דף ההזמנות יהיה ב:
   /booking/index.html
